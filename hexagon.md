Game Board
- Hexagon is placed on a hexagon grid.
- Players take turn, placing one tile, then the board is updated, then the next player's turn.

There are several different tiles:
- Empty tile (which might be unplaced or removed)
- Standard tile with 2x6 connectors, which might be connected or not
- Interacting tiles with 2x6 connectors, which do some actions after each players turn (like rotating).

There are also several connectors:
- Between tiles are tile2tile connectors
- On the tile are on-tile connectors
- At the boundary of the edge are dead-ends or OuterConnectors
Both might be teleporters.
There might be pickups at the connectors.


Game modes:

Standard game: (Single and multiplayer)
This is played on a roundish hexagon, with random tiles.
There are different winning conditions:
- Last man standing
- Maximal distance traveled
- Fastest peak velocity (longest distance traveled in a single turn)
This can be played in single player mode:
- Highscore
- Travel at least X steps on a partially filled board

Pick-up and deliver: (Singleplayer)
This is played on a partially filled board.
There is one active player and one (or several) inactive ones.
The inactive players need to be moved/connected to a target connector
